# SourceHut

Woodpecker comes with built-in support for SourceHut. To enable SourceHut you should configure the Woodpecker container using the following environment variables:

```ini
WOODPECKER_SOURCEHUT=true
WOODPECKER_SOURCEHUT_URL=YOUR_SOURCEHUT_URL
WOODPECKER_SOURCEHUT_CLIENT=YOUR_SOURCEHUT_CLIENT
WOODPECKER_SOURCEHUT_SECRET=YOUR_SOURCEHUT_CLIENT_SECRET
```
